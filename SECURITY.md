# Polityka bezpieczenstwa

## Zglaszanie luk bezpieczenstwa

**Nie zglaszaj luk bezpieczenstwa przez publiczne Issues.**

Zamiast tego wyslij email na adres podany w profilu autora lub uzyj [GitHub Security Advisories](../../security/advisories/new).

W zgloszeniu zawrzyj:
- Opis podatnosci
- Kroki do odtworzenia
- Potencjalny wplyw
- Proponowana poprawka (opcjonalnie)

Odpowiem w ciagu 48 godzin. Krytyczne luki beda naprawione w ciagu 7 dni.

## Zakres

Dotyczy:
- Kod wtyczki Ligase (PHP, JavaScript)
- Endpointy AJAX
- Przetwarzanie danych uzytkownika

Nie dotyczy:
- WordPress core
- Inne wtyczki/motywy
- Infrastruktura hostingowa
